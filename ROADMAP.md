# Roadmap

This roadmap lists public contribution areas. It is not a commitment to ship dates.

## Near Term

- Improve automated backend and frontend regression coverage for live/paper trading workflows.
- Add more feature-level screenshots and short demo clips for public onboarding.
- Expand OpenAPI-compatible API documentation for tenant-scoped admin endpoints.
- Add CI workflows for Maven tests, frontend build, Playwright smoke tests, and token-budget checks.
- Harden deployment docs for Oracle Cloud and generic Docker Compose hosts.

## Contributor-Friendly Issues

- Documentation examples for local Upstox sandbox setup.
- Feature doc diagrams for Intra Trade, Trading Scripts, Market Watch, and Option Chain.
- Small UI accessibility improvements in high-density trading tables.
- Additional frontend e2e coverage for empty/error states.
- Provider payload fixtures that avoid real account data.

## Not Planned By Default

- Live trading without explicit user confirmation.
- Committing broker secrets, account data, or production tokens.
- Replacing tenant isolation with single-user shortcuts.
